# 参考書ルート(診断×所持書の照合結果)

作成:2026-07-10 / 前提:浪人・私立専願(早稲田①明治②)/ 診断=特定クラスの知識欠落×未自動化(log.md参照)

## 判断の原則
1. 診断で実測された欠落クラス(A数量比較・B基本動詞・C機能チャンク・D記号・E切れ目・F語義検索・G段落構造)を埋める本を優先する
2. 役割が重複する本は1冊に絞り、残りは「使わない」と明示する(積んだ本を消化することは目的ではない)
3. 新規購入は不要。全クラスが所持書でカバーできる

## 時期の流れ
- **7〜8月**:上のセットで日課確立(初見月20〜25本+再読)。目標=「知識の穴で止まらない」状態
- **9〜10月**:The Rules 2完了 → 長文ポラリス1。明治過去問に初回接触。単語2冊目と文法発展の要否をここで判断
- **11月〜**:長文ポラリス2 → 早稲田過去問。過去問で出た欠落を欠落ノートに集約

## 1日の型(再掲・所持書に接続)
1. 初見1題(The Rules または プレノート):1回読み→段落ごとに言う→精読→欠落を7クラス分類→音読3〜5回
2. 再読:前日・3日前の本文(ぐんぐんBASIC含む)
3. 固定:1900を15〜20分、最前線1515を10分
4. 熟考下1〜2題。所要時間が長い日は1題で止める
5. 詰まった項目は大全・鉄壁を「引く」
- **最小構成(時間がない日)**:再読1本+1900の15分。ゼロの日を作らない

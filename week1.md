## 2. 診断読みプロトコル(長文1題の工程分解 = 表の「同」の中身)

| 工程 | 内容 | 目安 |
|---|---|---|
| ① 1回読み | 普段の速度。戻り読み禁止。段落末ごとに内容を口頭で言う(言えなければ「言えなかった」とだけメモして先へ) | 10〜15分 |
| ② 設問 | そのまま解く(Rulesは設問処理もルール化されているため飛ばさない) | 10分 |
| ③ 精読・答え合わせ | 解説と照合。初読の理解とのズレを文単位で特定。各文の根拠を「文内の証拠/常識」でラベル | 20〜30分 |
| ④ 欠落抽出 | ズレの原因を7クラス(A数量比較/B基本動詞/C機能チャンク/D記号/E切れ目/F語義/G段落構造)に分類し、欠落ノートに1行ずつ | 5〜10分 |
| ⑤ 引く | B・A・D系の欠落は真・英文法大全の索引、基本語の多義はターゲット1900→鉄壁の順で引いて核を確認 | 5分 |
| ⑥ 音読 | 意味が同時に浮かぶ速度で×5回(表の「×5」)。翌日×2、3日後×1 | 10〜15分 |

## 4. 語彙の再起動方式(1900)

| 項目 | 内容 |
|---|---|
| 基準 | 「見て1秒で語義が出る」か否かの2択で仕分け(書かない・悩まない) |
| 分量 | 1日200語を高速チェック→出なかった語だけ翌日再チェック(実質20分) |
| 開始位置 | 既習範囲の先頭から(現在地の申告があれば調整) |
| 多義語 | 長文で誤確定した語(adopt等)は鉄壁で核イメージを確認し、欠落ノートFクラスへ |

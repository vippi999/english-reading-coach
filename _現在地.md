# 英語長文読解｜現在地

## 目的

実際の初読・精読データに基づいて、英語長文で処理が止まる原因を特定し、参考書と日々の学習方法へ落とし込む。

## 済んだこと

- 2長文・31事例を記録し、28件を暫定分類した。母数が小さいため、比率と診断は未確定として扱う。
- 所持参考書の役割、暫定ルート、月間・第1週計画を作成した。
- このチャットを司令塔、別チャットを実読担当とする役割分担を決めた。
- 実読担当用の固定手順と、第1巡の指示を作成した。
- 実読担当はChatGPTモバイルを使い、GitHubの非公開リポジトリ経由で指示書を読む方針にした。
- `vippi999/english-reading-coach` を非公開リポジトリとして公開し、ChatGPT Codex Connectorにはこのリポジトリだけを許可した。ChatGPT側から `_現在地.md` の取得まで確認済み。

## 次の一手

- 通常のテキストチャットから `手順_実読チャット.md` と `指示_実読チャット.md` を読ませる。
- 実読担当は未読の The Rules 1 を3セッション実施し、`実読チャット報告_YYYY-MM-DD.md` を1つ作る。
- このチャットが報告を受け取り、`log.md`・`weakness_data.md`・計画類を更新して次の指示を出す。

## 参照パス

- `手順_実読チャット.md`: 実読担当の固定手順と報告書式
- `指示_実読チャット.md`: 司令塔から実読担当への現在の指示
- `log.md` / `weakness_data.md`: 読解事例と暫定集計
- `plan.md` / `route.md` / `week1.md`: 現時点の計画案
- `会話全記録_1.md`: 旧チャットの逐語記録
